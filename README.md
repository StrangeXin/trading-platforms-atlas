# Trading Platforms Atlas

> A searchable knowledge base of the global trading platform ecosystem — retail FX, institutional, equities, futures, crypto. History, business relationships, technical protocols, and the current landscape.
>
> 全球交易平台生态的可检索知识库 —— 零售外汇 / 机构 / 股票 / 期货 / 加密。历史、商业关系、技术协议、当前格局。

## Structure / 目录结构

Each language has the same six sections:

| Folder | English | 中文 |
|---|---|---|
| `01-history/` | Timeline + key origin stories | 时间线 + 起源故事 |
| `02-platforms/` | 20 platform deep-dives | 20 个单平台深度档案 |
| `03-architecture/` | Tech stack comparisons | 技术栈对比 |
| `04-relationships/` | Ownership / liquidity / white-label maps | 关系图谱 |
| `05-trends/` | Current trends & predictions | 当前趋势分析 |
| `06-research-notes/` | Source index | 原始资料索引 |

## How to use / 用法

- **Look up a platform**: start at `<lang>/02-platforms/<name>.md`, cross-reference via `<lang>/03-architecture/` and `<lang>/04-relationships/`
- **Trace a historical thread**: `<lang>/01-history/00-timeline.md` is the index; drill into the dated subchapters
- **Search**: it's all markdown — `grep` / `ripgrep` works across both languages

---

## Source priority / 信息来源优先级

1. Primary documents — regulatory filings, S-1 prospectuses, API docs, whitepapers
2. First-hand reporting — WSJ / FT / Bloomberg / Reuters
3. Industry blogs — Finance Magnates / FX News Group / LeapRate
4. Tech communities — Hacker News / Reddit r/algotrading / Stack Overflow
5. Third-party market data — Statista / Finance Magnates Intelligence / BIS

URLs are inline per chapter; consolidated index in `<lang>/06-research-notes/raw-sources.md`.
