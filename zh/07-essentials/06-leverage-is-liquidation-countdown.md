# 杠杆本质是爆仓倒计时

**单句论点**：杠杆不是"放大收益"的中性工具。它是一个严格按照数学规律，把账户推向 0 的加速器。给定 broker 的点差 + 偏见的 overnight swap，**几乎所有零售账户的预期终值是 0**，杠杆决定了到达 0 的速度。

## 一、直觉 vs 数学

**直觉叙事**：
> "杠杆让小资本也能参与大市场。100 倍杠杆 + 1% 盈利 = 100% 账户收益，复利下去就财富自由。"

**数学现实**：
- 杠杆对正收益和负收益**都放大**
- 但负收益的复利效应是**不对称**的（亏 50% 要赚 100% 才回本）
- 加上**每笔交易的点差 markup**（零售典型 0.5-2 pip）
- 加上**overnight swap 的系统性偏见**（broker 设定有利自己）
- **所有上述三项叠加后**，零售账户的**期望值是负的**
- 杠杆越高，达到期望终点（0）的时间越短

## 二、account lifetime 的数学推导

假设：
- 起始账户余额 $E_0$
- 每笔交易的期望收益率 $\mu$（零售典型 = -0.1% 到 -0.3%，因为点差 + 不利 swap）
- 每笔交易的波动率 $\sigma$（典型 = 1-3% per round trip）
- 每天交易次数 $n$
- 杠杆 $L$

**账户价值的随机漫步**模型（简化）：
$$E_t = E_{t-1} \cdot (1 + L \cdot r_t)$$

其中 $r_t$ 是单笔交易的回报，均值 $\mu$，方差 $\sigma^2$。

**账户爆仓的条件**：$E_t$ 跌破维持保证金（通常 = 起始的 20-50%），强制平仓后账户归 0。

**关键结论**（用 Geometric Brownian Motion 近似）：

达到完全爆仓的期望时间 $T$ 约为：

$$T \approx \frac{-\log(0.5)}{L \cdot (|\mu| + \sigma^2 / 2)}$$

代入典型零售参数（$\mu = -0.2\%$, $\sigma = 2\%$, 每天 5 笔）：

| 杠杆 $L$ | 每天波动 | 期望账户寿命 |
|---|---|---|
| 1:1 | ~2% | 几年（几乎不会爆）|
| 1:10 | ~10% | 6-12 个月 |
| 1:30（欧洲 ESMA 上限）| ~30% | 3-6 个月 |
| 1:100 | ~100% 理论值 | 2-4 个月 |
| 1:500 | - | **几周到几个月** |
| 1:1000 | - | **几天到几周** |

（注：高杠杆下"每天波动"大于 100% 的数学不成立 —— 爆仓概率每天接近 100%，所以账户通常第一天就亏完。）

## 三、Kelly 公式的边界

**Kelly 公式**（John Kelly 1956）给出在**已知边际** $b$、**胜率** $p$ 情况下的**最优仓位**：

$$f^* = \frac{bp - q}{b}$$

其中 $q = 1-p$ 是败率。

**Kelly 的核心论点**：
- **低于 Kelly**：你长期还是赢，但增长慢
- **等于 Kelly**：最快增长（但波动性高）
- **超过 Kelly**：**长期期望值为负**（不是慢慢赢，是必然输）

**典型零售 trader 的参数**（如果他胜率 55%，盈亏比 1.5:1，这已经是**专业水平**）：

$$f^* = \frac{1.5 \times 0.55 - 0.45}{1.5} = 0.25 = 25\%$$

即使是专业水平的 trader，Kelly 仓位也只有 25%。

**但普通零售交易者的真实参数**：
- 胜率 45-50%（低于对抽）
- 盈亏比 0.8-1.2（因为心理恐惧 / 贪婪让你早止损晚止盈）

$$f^* = \frac{1.0 \times 0.5 - 0.5}{1.0} = 0 \rightarrow \text{不该交易}$$

**即 Kelly 告诉你：仓位应该是 0**。但零售 broker 的 UI 强推你用 1:100 甚至 1:500 杠杆 —— **这是超 Kelly 仓位的 500 倍以上**。

## 四、"Half-Kelly" / "Quarter-Kelly" 实操智慧

专业 trader / quant 从不按满 Kelly 交易。业内共识：

- **Half-Kelly**（Kelly 的 50%）：风险回报平衡
- **Quarter-Kelly**（Kelly 的 25%）：保守
- **Twentieth-Kelly**（Kelly 的 5%）：极度保守，典型大基金做法

**为啥不用满 Kelly**：
- Kelly 假设你知道**真实**胜率 / 盈亏比 → 你不知道，只有估计
- 估计误差会让你实际过了 Kelly 而不自知
- 超 Kelly 的"长期必输"性质极其恐怖
- 所以工程上永远**打 2-20 倍安全余量**

**对应到零售**：
- 即使你估计自己"胜率 55% 盈亏比 1.5"
- Kelly 给 25%
- Quarter-Kelly 给 6%
- 但 MT4/MT5 的"一键下单"按钮默认**一手 = 账户的 10-100%**
- 你用了 broker 推的杠杆 → 直接超 Kelly 几十倍

## 五、杠杆放大的不是回报，是波动的不对称

**事实**：100% 盈利要 50% 亏损才抵消。数学上不对称：

| 亏损 | 需要多少盈利才回本 |
|---|---|
| 10% | 11.1% |
| 20% | 25% |
| 30% | 42.9% |
| 50% | 100% |
| 70% | 233% |
| 90% | 900% |

**杠杆放大后**：
- 小波动（±2%）在低杠杆下是普通振荡，在高杠杆下就是爆仓
- 每次爆仓后，你"回到起点"需要的盈利增速翻几倍
- **这个不对称性叠加上系统性 markup**，账户期望值严格负数

### 数字说话：1:100 杠杆下一个中等波动周

假设 EUR/USD 一周内 ±1.5% 波动（正常）。

1:100 杠杆下你的账户每天面临 ±100% × 0.5% = ±50% 的账户波动。

**5 天后的账户分布**（蒙特卡罗模拟）：
- **50% 账户已爆仓**（触及维持保证金）
- **30% 账户亏 30-70%**
- **15% 账户接近起始**
- **5% 账户盈利 50%+**

即使你是这 5% 的幸运儿，下一周同样的分布 → 连续 4 周还在 5% 俱乐部的概率 = 0.05^4 = **0.000625%**。

这就是为啥**零售 broker 数据显示 12 个月活跃率 <10%**。不是 trader 技能问题，是数学问题。

## 六、为啥 Broker 爱卖高杠杆

从 broker 视角，杠杆不是"给用户的工具"，是**用户生命周期加速器**：

| 杠杆 | 账户寿命 | Broker 收入率（每 $1 入金）|
|---|---|---|
| 1:10 | 12 个月 | 稳定慢，年化 10-20% |
| 1:100 | 3 个月 | 快爆 → 再入 → 再爆，年化 40-80% |
| 1:500 | <1 个月 | 极速循环，年化 100%+ |

Broker 的 LTV / 时间 在高杠杆下最大化。**他们明知道高杠杆让用户更快亏完** —— 这是 feature，不是 bug。

**ESMA 2018 年为啥砍到 30:1**：
- 研究发现 > 100:1 杠杆下，12 个月失败率**接近 100%**
- 30:1 下降到 74-89%（已经很高，但"只有大部分亏完"而不是"全部亏完"）
- 政治上是妥协 —— 真正保护消费者会砍到 5:1，但行业游说成功阻止

## 七、高杠杆的心理学陷阱

### 陷阱 1：小资本的"翻身"幻想

- 用户：$1000 本金 + 1:500 杠杆 = $500,000 名义头寸
- 感觉："只要 EUR/USD 动 1%，我就赚 $5000 = 500%"
- 现实：EUR/USD 0.4% 反向波动 = 爆仓
- 心理层面：用户以为自己在"以小博大"，实际是在"以小博 0"

### 陷阱 2：Survivor bias in marketing

- Broker 营销素材：某 trader 用 $1000 赚到 $1M
- 这是真实的 —— 但忽略了 99.99% 同样试的人归零
- 用户只看到赢家图片，没看到亏家群组里 100 万人的沉默

### 陷阱 3：每次爆仓后的"差一点"心理

- 用户爆仓 → 心理计算"如果我当时止盈早一步就赢了"
- 下次再入金 → 这次要更小心 → 但杠杆还是 1:500 → 再爆仓
- **沉没成本 + 证明自我**驱动持续入金
- Broker 的 IB / 客服在这个阶段**主动营销 bonus / 再入金奖励**

## 八、实操建议

**最严酷的真相**：如果你是零售 trader，**任何超过 5:1 的杠杆都是在加速你账户归零**。

### 杠杆使用准则

| 杠杆 | 场景 | 评价 |
|---|---|---|
| 1:1（现货） | 长期持仓 / 价值投资 | ✅ 安全 |
| 1:2-3 | 有明确止损的波段交易 | ✅ 可接受 |
| 1:5 | 有经验 + 严格风控 | ⚠️ 上限 |
| 1:10 | 专业对冲基金用 | ⚠️ 需要专业风控 |
| 1:30+ | 零售"赌命" | ❌ 长期必输 |
| 1:100+ | 赌博 | ❌ 不是"投资" |
| 1:500+ | 纯炒作营销 | ❌ 数学上必爆 |

### 关键心法

1. **杠杆是成本，不是工具**。每用 1 倍杠杆，你就多交一次"波动税"
2. **时间是你的朋友（低杠杆）或敌人（高杠杆）**。长期持仓的复利对低杠杆有利，对高杠杆有害
3. **仓位来自 Kelly，不是来自"UI 上的按钮"**。算出你的 Kelly，取 Quarter-Kelly
4. **如果你亏损 3 个月，不是运气，是杠杆**。降杠杆到原来的 1/3 再评估
5. **如果你连续盈利 3 个月，也不是技能**，是运气。正好此刻降杠杆别膨胀

## 九、数据与历史

### ESMA 2018 前后数据对比

欧盟零售 CFD 杠杆从 无限 → 30:1 major / 20:1 indices / 10:1 商品 / 5:1 小币对 / 2:1 加密

- 2017（砍之前）：零售损失率 **89%**
- 2019（砍之后）：零售损失率 **74%**

**砍杠杆救了 15% 的人**。如果砍到 5:1，估计能救 30-40%。但没砍 —— 行业游说成功。

### 2015 SNB 瑞郎事件：杠杆的极端案例

- 瑞郎 30 秒跳升 20%
- 用 1:100 杠杆 long EUR/CHF 的零售账户 → 瞬间欠 broker 钱（负余额）
- FXCM 全美最大零售破产（穿仓 $225M）
- Alpari UK 直接倒闭
- 用户的"赢利"被取消（broker 用 TOS 12.3.b"非正常市场"条款），"亏损"被追讨
- 监管由此推出"负余额保护"（broker 不能向客户追讨超过余额的亏损）

### Robinhood 加密 2022 年

- 零佣金 + 24/7 加密
- 用户用"margin" 买入 Doge / Shib（实质杠杆）
- 2022 Q2 市场崩盘 → 用户集体爆仓
- Robinhood 股价跌 70%（客户损失 = 公司营收高 = 但客户没了）

**每一次杠杆普及 + 下跌周期都复现同样模式**。

## 十、如果你非要做"交易"

1. **现货持仓**，不加杠杆 —— 你就是"持股 / 持币者"，长期有效
2. **最多 2-3 倍杠杆** —— 对应 Kelly 的 Quarter 对应典型边际
3. **总账户风险的 1-2% 一次 trade** —— 这是所有专业风控的底线
4. **止损比止盈重要** —— Asymmetry 里止损保护了你
5. **退出职业化的勇气** —— 90%+ 的零售 trader 退出后生活更好
