# 监管本质是行业的护城河工具

**单句论点**：ESMA / FCA / ASIC / CFTC 不是"消费者保护机构"——是**按市场准入费收租**的行业守门人。每次"重大监管"都提高了合规成本，把小玩家挤出去，把在位者的利润率锁定。

## 一、直觉叙事 vs 结构现实

| 叙事 | 现实 |
|---|---|
| 监管保护散户 | 监管设定了"合法伤害散户"的速率上限 |
| 严监管 = 安全 | 严监管 = 高门槛 = 大品牌寡头 |
| 监管罚款让行业改过 | 罚款金额 << 违规期间的收益；是做生意成本 |
| 监管防止下一个 FTX | 每次 FTX 之后都有新"更严"监管；下一个 FTX 还会来 |
| "Regulated" 就安全 | Regulated 只意味着付了年费 |

## 二、MiFID II：欧盟监管"升级"的真面目

**背景**：2018 年 1 月生效的 MiFID II（Markets in Financial Instruments Directive II）是欧盟金融监管史上最大一次"升级"。官方目的：保护零售投资者、增加透明度。

**实际影响**：

### 合规成本飙升

| 角色 | MiFID I 合规年成本（欧元）| MiFID II 合规年成本 |
|---|---|---|
| 小型 CySEC broker | ~€10,000 | ~€200,000-500,000 |
| 中型券商 | ~€50,000 | ~€800,000-1,500,000 |
| 大型投行 | ~€500,000 | ~€3,000,000-8,000,000 |

**~10 倍成本跳升**。业内称 "MiFID II compliance tax"。

### 合规的具体动作

- 报告每笔交易到中央仓库（ARM）
- 每个 trader 要 LEI（Legal Entity Identifier）
- 每个订单的"最佳执行"证明
- 录音电话 + 存档 5 年
- "Product Governance"文件每个产品
- Transaction Cost Analysis 月度报告
- ...

这些中**绝大多数对散户毫无保护价值** —— 是给机构 audit trail 用的。

### 结果

**2018-2020 欧盟零售 FX broker 数量**：
- 2017：~400+ 家持牌
- 2020：~280 家
- 2023：~230 家

**倒掉 / 并购的 170 家**基本全是小玩家。头部 10 家（IG、CMC、Plus500、Saxo、Interactive Brokers 欧洲、Pepperstone、XTB、eToro、Admiral Markets、OANDA）**市场份额从 55% 涨到 ~80%**。

**"严监管保护散户"** → 实际上**保护了寡头**。

## 三、牌照作为"进入税"

### 各辖区零售 FX broker 牌照成本（2024 业内估算）

| 辖区 | 初始申请 + 资本 | 每年合规成本 | 时间线 |
|---|---|---|---|
| **CySEC（欧盟最常见）** | €500K + €750K 资本金 | €200K+ | 9-18 个月 |
| **FCA（英国）** | £1M+ + £730K 资本金 | £500K+ | 12-24 个月 |
| **ASIC（澳大利亚）** | A$500K + A$1M 资本金 | A$300K+ | 6-12 个月 |
| **NFA / CFTC（美国）** | $10M+ + $20M 资本金 | $2M+ | 18-36 个月 |
| **FINMA（瑞士）** | CHF 10M+ + CHF 10M 资本金 | CHF 2M+ | 24-36 个月 |
| **IFSC Belize / Vanuatu** | $5K + $100K 资本金 | $20K | 2-3 个月 |

**你读对了**：美国 CFTC 的有效"进入税"约 **$30M +** 每年 $2M 运营。这不是"保护消费者"的合理门槛 —— 是**把所有小玩家从市场清除**。

### 美国零售 FX broker 的寡头格局

因为 CFTC / NFA 门槛极高，**美国零售 FX 市场只有 3 家合规 broker**：

- **OANDA US**
- **FOREX.com**（Gain Capital 旗下）
- **IG US**（2019 年才拿到牌照）

**整个美国 3.3 亿人口，只有 3 家外汇券商**。对比欧洲（200+）、澳大利亚（~50）、亚洲（500+）。

这不是因为美国监管"更严格"，是因为监管**故意设定成寡头市场**。在位者不希望更多竞争者，监管配合他们限制进入。

## 四、旋转门：监管到行业的职业路径

### 典型职业轨迹

```
FCA 初级研究员 (年薪 £40K)
    ↓ 3-5 年
FCA 高级 policy advisor (年薪 £80K)
    ↓ 3-5 年
跳到投行合规部 (年薪 £250K+)
    ↓
成为业内 lobbyist / consultant
```

**案例（公开记录）**：
- 前 FCA CEO Andrew Bailey 2020 年回到 Bank of England 做总裁
- 前 ESMA 主席 Steven Maijoor 2021 年跳到荷兰央行
- 前 SEC 主席 Jay Clayton 2021 年加入 Apollo Global Management（PE）
- 多位 CySEC 高级审核员跳到塞浦路斯大 broker 做合规长

**为啥重要**：

- 监管人员在任上知道**"以后会被什么公司雇"**
- 理性选择：**不要太严厉对未来雇主**
- 理性选择：**提高小玩家门槛**（因为大玩家是未来雇主）
- 这不是"腐败"，是**激励结构**

### 学术界论证

- **"Regulatory Capture"** 概念由 George Stigler 1971 提出（芝加哥大学，诺贝尔奖）
- 后续大量实证研究（金融、能源、医药）证明监管被行业俘获是规律不是例外
- 2008 金融危机后关于 SEC / Fed 的"capture"研究是显学

## 五、"Negative balance protection" 看似保护散户，实际保护行业

ESMA 2018 强制零售 CFD broker 不能向客户追讨超过余额的损失。这个规则看起来**明确保护散户**。

**但深层看**：

1. **大多数主流 broker 已经自愿实施过**（2015 瑞郎事件后为吸引客户）
2. **没有这条规定**，极端事件下 broker 向散户追讨导致的**社会舆论 / 政治压力**会引发更严格的立法
3. **强制负余额保护**让 broker：
   - 更放心地推高杠杆（反正散户最多输到 0）
   - 保险公司或 reinsurance 处理尾部风险
   - **避免下一次 politically embarrassing 事件**
4. **散户实际受益**：从"极罕见的负余额追讨"→"没有"。但同时接受**杠杆上限 30:1 而不是 5:1**

**本质**：监管用"可见保护"（负余额）换取了"不可见扩张"（允许 30:1）。零售的 74% 继续亏损率并未因此改善。

## 六、CFTC vs 非传统产品：期权 / 加密 / Prop Firm

### 为啥 Nadex 能存在

- Nadex（原 HedgeStreet）是 CFTC 监管的零售"binary options"
- **全美合规**唯一 retail binary
- 逻辑：HedgeStreet 2000 年代花了巨额合规费用**先进场**
- 后来 CFTC 禁止 broker 向散户提供 binary options —— **但给了 Nadex 豁免**
- Nadex 垄断美国零售 binary ~15 年
- 2022 年被 Crypto.com 以 $216M 收购

**这是典型监管套利套利**：**不是** Nadex 产品更安全，**是** Nadex 是"合规圈内的第一个"。

### 为啥 Robinhood 2021 受罚

- GameStop 事件期间 Robinhood 限制交易
- SEC 调查几个月 → $70M 罚款
- Robinhood 正常继续运营
- $70M 罚款 = Robinhood 6 个月正常广告预算
- **这是交税，不是处罚**

### MyForexFunds（2023）CFTC 起诉的"政治时机"

- MFF 2020-2023 迅速崛起，2023 Q1 月营收估计 $10M+
- CFTC 2023 年 8 月突然起诉 $310M "fraud"
- **巧合**：**同时** CFTC 正在为其他"监管范围"（如 crypto）争取预算
- 一次"重大执法行动"有利监管自身 political standing
- 2025 年 5 月 Special Master 报告：**CFTC 调查程序有重大瑕疵**，部分指控被驳回
- **MFF 已经倒闭**，无法追索"损失"

这就是监管执法的**选择性 + 政治化**本质。MFF 确实有问题，但为啥选 MFF 而不选其他 50 家类似 prop？因为 MFF 规模大到**"适合作执法案例"**。

## 七、辖区套利：监管"严"和"松"的真实含义

### 典型大 broker 的多辖区策略

一个像 Exness 这样的头部 broker，持有的牌照：

- **CySEC**（欧盟 + 亚洲**部分**客户）：30:1 杠杆
- **FCA**（英国）：30:1 + 强合规披露
- **FSCA**（南非）：500:1
- **CMA**（肯尼亚）：无限杠杆
- **FSC Mauritius**：无限杠杆
- **FSC BVI**：无限杠杆

**客户注册时的路由逻辑**：
```
如果 IP / 身份证 = 欧盟居民 → CySEC 实体
如果 IP / 身份证 = 英国 → FCA 实体
如果 IP / 身份证 = 中国 / 其他"高杠杆接受地区" → Mauritius / BVI 实体
```

**关键**：
- 全部是**同一集团**运营
- 品牌相同、界面相同、团队相同
- 只有**法律实体和允许的规则**不同
- 营销按"持有多家 Tier-1 牌照"宣传公信力
- 高杠杆用户默认被路由到低监管实体

### 监管为啥容忍这种辖区套利

- 欧洲监管机构只管自己的管辖区内客户
- 不想"推走"总部在欧洲的品牌
- 如果 CySEC 硬性要求 Exness 所有子公司都按 30:1 —— Exness 会直接搬总部到迪拜
- 迪拜监管更松 + Exness 仍然可以服务欧洲客户**通过法律漏洞**
- CySEC 失去监管权 + 税收 + 就业

**结果**：表面"严监管"的辖区接受了"集团主体监管 + 子公司套利"的默认规则。这是游说、税收、就业的妥协。

## 八、监管"真正在保护"的对象

按实际行为判断，监管保护的优先级：

1. **监管自身的 political standing**（防止大规模 scandal 引发国会 / 议会追责）
2. **在位大品牌的利润率**（setting 维持 oligopoly）
3. **监管员工未来的就业机会**（不要得罪潜在雇主）
4. **外汇 / CFD 市场的系统稳定**（防止像 2008 那样连锁破产）
5. **散户绝对亏损不要变 scandal**（给个 "-max loss = deposit" 的上限）
6. ...遥远地**下面才是散户不被诈骗**

零售散户**在优先级 6-7**。"消费者保护"叙事只是包装。

## 九、历史类比：美国 20 世纪金融监管

### 1913 Federal Reserve Act

- 创立的背景：**1907 年大恐慌** + JP Morgan 个人救市不可持续
- 表面：保护储户 / 稳定银行业
- 实际：**由最大的几家华尔街银行（Morgan、Rockefeller、Kuhn Loeb）起草**
- 结果：巩固了这些银行的核心地位，小银行被系统性清除

### 1933 Glass-Steagall

- 大萧条后的"保护散户"监管
- 强制商业银行和投行分离
- **受益最大**：之前靠综合业务的小银行被迫选边，大银行保持规模优势
- 1999 年 GLB Act 废除 —— 大银行游说成功

### 2010 Dodd-Frank

- 2008 危机后的"大保护"法案
- 合规成本让小地区银行倒闭 ~1000 家
- 大银行"Too Big to Fail"的地位**被法律明确承认**（SIFI 机构）
- 真正的 moral hazard 被**制度化**

**每次"保护散户"的监管升级，结果都是寡头化**。这不是意外，是结构。

## 十、给个人的认知框架

### 理解"Regulated" 的真实意思

- **"FCA Regulated"** = 这家公司付了 £500K+ 年费给 FCA
- **并不**意味着公司做的交易对你更公平
- **并不**意味着你的资金 100% 安全（SIPC / FSCS 有上限）
- **确实**意味着公司有**成熟的法律合规架构**（不会随便跑路）
- **确实**意味着**爆雷成本**对公司更高（更有动机不做极端违规）

### 选择 broker 的实际指南

**"Regulated"** 是必要但不充分条件：

1. **看监管级别**：FCA / ASIC / FINMA / FCA US > CySEC > FSCA > 塞舌尔 / BVI
2. **看公司结构**：只有一个实体 vs 多层辖区套利
3. **看披露**：公司财报 / 年度审计是否公开
4. **看历史**：是否经历过极端市场事件 + 如何处理（2015 瑞郎、2020 疫情、2022 加密崩盘）
5. **看 LP 披露**：真 A-book broker 会列出具体 LP 银行
6. **看客户反馈的 pattern**：个别差评正常；系统性同一 pattern（提现难、突然封号）是红灯
