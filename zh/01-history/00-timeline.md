# 电子交易大事记：1970s–2026

按时间顺序梳理电子交易、零售外汇、加密市场的关键事件。每条带日期 + 一句话说明。

---

## 1970s — 奠基

| 日期 | 事件 |
|------|-------|
| 1971-02-08 | **NASDAQ 上线**，全球第一个电子股票市场；起初只是个电子公告板，每天更新一次报价，下单还得打电话。 |
| 1976 | **NYSE DOT（Designated Order Turnaround）** 上线，小额股票订单可以电子路由到场内 specialist。 |

## 1980s — 基建成熟

| 日期 | 事件 |
|------|-------|
| 1982 | **Timber Hill Inc.**（Interactive Brokers 前身）开始写股指期货和期权定价系统。Thomas Peterffy 1983 年把第一批手持交易终端带进 AMEX 场内。 |
| 1984 | **NASDAQ SOES**（Small Order Execution System）推出，首次实现小额零售订单的电子执行。 |
| 1984 | **NYSE SuperDOT** 取代 DOT；单笔可处理 10 万股，容量扩 1,000 倍。 |
| 1986-10-27 | **英国"Big Bang"** 放松管制，废除固定佣金、开放伦敦交易所给外资，欧洲股票快速电子化。 |
| 1988 | 1987 年股灾后，**NASDAQ SOES 对所有做市商强制**，逼着全行业上电子化。 |

## 1990s — 电子市场站稳

| 日期 | 事件 |
|------|-------|
| 1990 | **NASDAQ SelectNet** 推出，支持超出 SOES 上限的大单电子交易。 |
| 1992 | Fidelity 的 Robert Lamoureux 和 Salomon Brothers 的 Chris Morstatt 搞出 **FIX Protocol 1.0**；第一个机器可读的股票订单协议。 |
| 1993 | **Interactive Brokers LLC** 作为美国 broker-dealer 注册成立，把 Timber Hill 的算法做市基础设施开放给外部客户。 |
| 1996 | **OANDA Corporation** 在温哥华 / 纽约成立；互联网时代最早一批 FX broker，做实时汇率数据起家，后来做零售现货 FX。 |
| 1997 | **REDIBook ECN** 上线（后来被 NYSE 吸收），把盘后电子股票交易带给零售 + 机构。 |
| 1998 | **SEC Regulation ATS** 允许 ECN 以交易系统身份注册，正式给传统交易所外的电子场地开了合法通道。 |
| 1999 | **Archipelago（ArcaEx）** 和 **Island ECN** 崛起，预示交易所级别的电子竞争。 |
| 1999 | **FXCM** 成立，互联网零售 FX 第一波之一，给散户做线上交易。 |

## 2000s — 零售 FX 大爆发 + 算法交易

| 日期 | 事件 |
|------|-------|
| 2000 | **MetaQuotes Software** 在俄罗斯喀山由 Renat Fatkhullin 创立；首款产品 FX Charts，早期 Web 版零售 FX 看盘工具。 |
| 2001 | **FIX 4.2** 发布；事实标准，二十多年后仍在广泛使用。 |
| 2002 | MetaQuotes 发布 **MetaTrader 3**，加期货 + MQL II 算法脚本语言。 |
| 2003 | **MetaTrader mobile**（CE 和 Palm 版）发布；MetaQuotes 提前三年进入移动交易。 |
| 2005-07-01 | **MetaTrader 4（MT4）上线**；架构完全重写，MQL4 + Expert Advisors + Strategy Tester。之后 15 年零售 FX 的统治级平台。 |
| 2006 | **FIX Trading Community** 作为非盈利标准组织成立，接管 FIX 协议的演进。 |
| 2007 | **eToro** 在特拉维夫成立（原名 RetailFX），2010 年把社交 / 跟单交易带进零售 FX。 |
| 2007-07-09 | **NYSE Euronext 合并**完成，第一个跨大西洋交易所集团诞生；大电子市场开始整合。 |
| 2008 | **Plus500** 成立；100% 线上 CFD 平台在以色列发布，扩到欧洲，全程无人工 broker。 |
| 2009-01-03 | **比特币创世区块**由中本聪挖出；比特币网络正式运行。 |

## 2010s — 高频、加密、Prop Firm

| 日期 | 事件 |
|------|-------|
| 2010-05-06 | **闪电崩盘**：DJIA 几分钟暴跌 ~1,000 点，HFT 反馈环是祸根；引爆全球关于算法市场结构的辩论。 |
| 2010-06-01 | **MetaTrader 5（MT5）上线**；多资产平台支持 FX、股票、期货，MQL5 + 更强回测。 |
| 2010-07-18 | **Mt. Gox** 开始做比特币交易所（最早是 Magic: The Gathering 卡牌交易站）。 |
| 2010 | **美国 CFTC** 把零售 FX 杠杆限制到 50:1（主要货币对）和 20:1（其他），重塑美国零售 FX 格局。 |
| 2011-04 | **cTrader** 由 Spotware Systems（塞浦路斯）发布；FxPro 是首家接入的 broker。 |
| 2011 | **TradingView** 成立（正式注册 2011 年）；早期云端看盘平台，2013 年被 TechStars 选中。 |
| 2012 | **IEX**（Investors Exchange）由 Brad Katsuyama 成立，反击掠食性 HFT；Michael Lewis 2014 年《Flash Boys》带火它。 |
| 2013 | **TradingView** 在 TechStars 期间签下 CME Group 和 Microsoft 数据合同；走向大众化看盘。 |
| 2013-04 | **比特币首次突破 $200**；Mt. Gox 占全球 BTC 成交 70%+。 |
| 2014-02-07 | **Mt. Gox 停止比特币提现**，随后破产；披露丢失 850,000 BTC（约 $473M）。触发首波加密所监管（日本）。 |
| 2015 | **FTMO** 在布拉格由 Otakar Šuffner 和 Marek Vašíček 创立；把"出资交易员挑战"模式做通。 |
| 2016 | **Nasdaq 收购 ISE**、完成收 Chi-X Canada；交易所整合潮延续。 |
| 2017-07 | **Binance** 由赵长鹏（CZ）在一次 $15M ICO 后成立；上线 6 个月内成为全球成交量第一的加密所。 |
| 2017 | **ICO 泡沫**：基于以太坊的代币发行募资数十亿美元；SEC 7 月发警告；2018 年初市值冲上 $800B+。 |
| 2018-05 | **TradingView** 完成 $37M A 轮，Insight Venture Partners 领投；收购 TradeIT（broker 下单中间件）。 |
| 2018 | **OANDA** 被私募 CVC Capital Partners 收购；加速数字平台投入。 |
| 2019-09-05 | **BUSD 上线**，Binance + Paxos 合作；美元背书的稳定币锚定 Binance 生态。 |
| 2019 | **Uniswap v1** 在以太坊主网上线；AMM DEX 模型挑战中心化交易所。 |

## 2020s — 加密成熟、Prop Firm 整肃、平台战争

| 日期 | 事件 |
|------|-------|
| 2020 | **COVID-19 疫情**全球零售交易爆发；PFOF 争议升温；Robinhood、eToro、prop firm 用户数空前增长。 |
| 2021-10 | **TradingView** 在 $298M 融资（Tiger Global 领投）中估值 $3B；成为全球零售看盘霸主，日活百万级。 |
| 2021 | **FTX** 估值顶峰 $32B；Sam Bankman-Fried 成为加密圈最风光的人物之一。 |
| 2022-09-23 | **Apple 全球下架 MT4 / MT5**；据猜与 MetaQuotes 俄罗斯背景 + 西方制裁挂钩。MetaQuotes 否认。 |
| 2023-03 | **Apple 重新上架** MT4 / MT5，前后大概 6 个月。 |
| 2022-11 | **FTX 暴雷**：~$8B 客户资金失踪；Sam Bankman-Fried 被捕；触发全球加密监管风暴。 |
| 2023-08 | **MyForexFunds（MFF）被关**：CFTC 和安省证监会冻结这家多伦多 prop firm 资产；指控 $310M 欺诈。案子后来崩盘——CFTC 被判赔 $3.1M 律师费。 |
| 2024-01-10 | **比特币现货 ETF 获 SEC 批准**（BlackRock、Fidelity 等）；BTC 机构化的里程碑。 |
| 2024-02-06 | **True Forex Funds** 无预警丢 MT4/MT5 license；MetaQuotes 整肃 prop firm 第一个公开牺牲品。CEO Richard Nagy："MetaQuotes 事先没给任何警告。" |
| 2024-02-11 | **Purple Trading** 在 MetaQuotes 施压下终止多家 prop firm（Funded Engineer、AquaFunded、Goat Funded Trader、The Funded Trader、Skilled Funded Trades）。 |
| 2024-02-14 | **Blackbull Markets** 终止 Funding Pips；CEO Anish Lal："我们被迫立即关停 Funding Pips。" MetaQuotes 整肃公开化。 |
| 2024 | 全球 **~80–100 家 prop firm 倒闭**；MetaTrader 在 prop firm 市场份额 9 个月内从 48% 跌到 24%。DxTrade、MatchTrader、TradeLocker、cTrader 分到蛋糕。 |
| 2024-01 | **Oleg Mukhanov** 出任 TradingView CEO；直连 broker 超过 100 家。 |
| 2025-Q1 | **MetaQuotes 停发所有新 MT4 license**；MT4 走向 EOL。存量 broker 继续用。 |
| 2025-07-01 | **MetaQuotes 停止支持**旧版 MT4 / MT5；强制最低 build 更新。 |
| 2025-12-01 | **FTMO 完成收购 OANDA**（从 CVC 手中）；第一家拥有多司法辖区持牌 broker 的大型 prop firm。8 个月拿下 5 个监管批准。 |
| 2026 | **Prop firm 监管压力升级**：英国 FCA、欧盟 ESMA、美国 CFTC 都在审查 funded challenge 是不是受监管的金融活动。 |

---

## 几条主线

1. **民主化曲线**：交易所场内 → ECN → 零售互联网 broker → 社交跟单 → prop firm 挑战 → AI 评分信号。
2. **平台从整合到破碎**：MT4 统治零售 FX 15 年；TradingView + 替代平台在侵蚀它的护城河。
3. **监管滞后**：每一次创新（HFT、零售 FX、加密、prop challenge）都比清晰的监管早 5–10 年。
4. **加密 / 传统金融融合**：比特币 ETF、prop firm 买持牌 broker（FTMO/OANDA）、交易所加法币进出通道。
