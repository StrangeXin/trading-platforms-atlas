# Plus500

以色列 / 塞浦路斯系 CFD broker 标杆。LSE 上市的"零售 CFD + 营销驱动"模板，90% 营收来自 B-book 的净收入。是整个"CySEC 系 CFD broker"商业模式最成功的公开案例。

## 基本信息

- **创立**：2008 年，海法（以色列）
- **创始人**：6 位 Technion（以色列理工）校友
- **监管**：CySEC（塞浦路斯主）+ FCA（英国）+ ASIC + MAS（新加坡）+ FSCA（南非）等 8 + 辖区
- **总部**：英国伦敦（2020 上市后搬）
- **LSE 上市**：2013 年，ticker PLUS
- **市值**：2024 年约 **£1.5-2B**（波动大）
- **员工**：~700（2024）
- **2024 活跃客户**：~425,000
- **2024 营收**：$720M
- **2024 净利润**：$294M（**40% 净利率**）

## 商业模式：最极端的 B-book 标本

### 纯 CFD 玩家

- **100% CFD**，不做股票 / 期货 / 基金
- 约 2,800 个 CFD 品种（indices / 加密 / forex / 大宗 / 个股 CFD / ETF CFD）
- 统一风险引擎 + 统一 B-book

### 营收结构（2024）

| 来源 | 占比 |
|---|---|
| **B-book customer losses（客户亏损的净头寸归 Plus500）** | ~**90%** |
| Spread 净值（在 LP 报价上加的 markup）| ~5% |
| Overnight swap | ~5% |

**90% B-book 意味着**：Plus500 基本**不做对冲**。当客户在做多 BTC，Plus500 在客户的每一笔多单**内部持空头**。市场动，Plus500 资产负债表跟着动。

### 为啥能活下来

**Plus500 的秘密**：**世界级的 B-book 风控系统**。

- 自研风险管理平台（代号 "Plus500 Trading Technology"）
- 实时监控客户头寸总净敞口
- 自动调整"展示给客户的 spread" → 让客户止损 / 平仓
- 关键事件（如瑞郎黑天鹅）**机器学习模型**预测调整杠杆 / 点差
- 当全球净头寸超阈值 → Plus500 自己去市场做反向对冲

2015 瑞郎事件时：
- 大多数 B-book broker（FXCM、Alpari、FXDD 等）负余额 hit 自身资本
- **Plus500 完好无损**（事后审计披露内部对冲及时触发）
- 这就是为啥它能 LSE 上市

### 客户获取机器

- **营销是核心成本**
- 2024 年 Plus500 花了 **~$160M 营销**（占营收 22%）
- 比较：IG Group 营销成本仅 ~5% 营收
- Google Ads FX / CFD 关键词 + Meta 广告 + YouTube 赞助 + 体育冠名（2018-2023 Atletico Madrid 主赞助商）
- 每个新客户成本（CAC）**$200-400**
- 平均新客户 LTV **$1,200-1,500**

### 无 FX 现货 + 无股票 → 无监管竞争

- CFD 是"合约"，不是"标的资产"
- Plus500 不需要 broker-dealer 牌照
- **不需要** 证券交易所连接
- **不需要** 第三方托管
- 纯粹是 CySEC CFD license 下的游戏

## 为啥是 CFD broker 模板

### 1. 100% 数字原生

- 成立就是 web-only
- 没有任何线下 / 电话业务
- 客户支持全部 live chat + email

### 2. 极简 UI

- 自研的 Plus500 平台（非 MT4 / MT5）
- 1 屏搞定开户 + 入金 + 交易
- 没有 EA、没有复杂指标、没有信号市场
- **故意做成低摩擦成交**

### 3. 营销 + 风控的双轮驱动

- 营销拉人（成本 22% 营收）
- 风控让 B-book 稳定盈利（90% 营收来自客户亏损）
- **其他所有东西都是配角**

## 2013-2024 股价和业绩周期

CFD broker 的营收**反周期** —— 市场波动越大越赚。

- **2017**：比特币牛市 + 加密 CFD 爆发 → 营收 $437M
- **2018**：ESMA 砍杠杆（30:1）→ 营收下滑
- **2019**：继续调整
- **2020**：**疫情市场大波动** + 新客户涌入 → 营收飙到 **$872M**（史上最高）
- **2021**：GME 事件 + 加密狂热 → 营收 $718M
- **2022**：加密熊市 → 营收 $832M（波动率仍高）
- **2023**：市场平静 → 营收 $685M
- **2024**：恢复增长 → 营收 $720M

**典型 CFD broker 季度股价 = 市场波动率函数**。这不是"投资机会"，是**对波动性的 beta**。

## 监管挑战

### 2015 英国 FCA 警告

- FCA 发布"CFD provider"行业警告，特别提到 Plus500
- 指控：客户留存率和赢利信息披露不足
- Plus500 回应：增加风险提示 + 改进披露
- **业务未受实质影响**

### 2018 ESMA 杠杆砍到 30:1

- Plus500 约 50% 营收来自欧盟
- 短期冲击严重（2018 营收下降 ~20%）
- 长期 Plus500 **适应并恢复** → 高风控系统受益于更严监管（小 broker 被挤出 → Plus500 市场份额扩大）

### 2024 澳大利亚 ASIC 改革

- ASIC 改善 CFD 产品披露规则
- Plus500 积极配合
- 是"合规适应性最强"的 broker 之一

## 产品细节

### 主要交易品种

**Indices CFDs**（营收占 40%+）：
- US 500, NAS 100, DE 40, UK 100 等

**Individual Stocks CFDs**（营收占 20%）：
- Tesla, Apple, Nvidia, AMD 等高波动热门股
- Plus500 **不实际持股** —— CFD 合约 vs broker

**FX CFDs**（营收占 15%）：
- EUR/USD, GBP/USD, AUD/USD, USD/JPY 等
- 相对窄 spread (1-2 pip)

**Crypto CFDs**（营收占 15%）：
- BTC, ETH, DOGE, SHIB 等
- **监管套利**：不是真加密所，是合约
- 欧洲用户能做加密 CFD，**真加密交易所** Binance / Coinbase 在欧洲有限制

**Commodities + Bonds CFDs**（营收占 10%）：
- 金、银、原油、债券期货 CFDs
