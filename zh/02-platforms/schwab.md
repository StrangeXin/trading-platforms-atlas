# Charles Schwab

**美国零售券商体量之王**。2024 年 AUC（Assets Under Custody，托管资产）**~$9.9 trillion**（万亿）—— 是 Robinhood AUC 的 **50 倍**。1971 年创立，开创美国"discount brokerage"模式，2020 年收购 TD Ameritrade 后成为**绝对寡头**。

## 基本信息

- **创立**：1971 年，旧金山
- **创始人**：Charles R. Schwab
- **上市**：NYSE: SCHW
- **总部**：西特普劳斯（Westlake）德克萨斯（2019 搬离旧金山）
- **员工**：~35,000（2024）
- **2024 AUC**：**$9.9 trillion**（第一大美国零售 broker，第二大 Fidelity ~$11T AUM）
- **2024 营收**：$19.6B
- **2024 净利润**：$5.9B
- **客户账户**：~36 million

## 历史与创立

### 1971-1975：电话券商起步

- **Charles R. Schwab**（生于 1937）在加州创立"First Commander Corporation"（后改名 Charles Schwab & Co.）
- 1971 年 **SEC 允许佣金自由定价**前，所有券商收 fixed commission（$65 / trade 起）
- Schwab 在 **1975-05-01 Mayday**（SEC 正式自由化当天）立即**降到 $8**
- **发明了"discount brokerage"** —— 不提供研究建议，只做执行，收低佣金

### 1975-1990：扩张

- **1976**：第一家零售分店（旧金山）
- **1980**：首家提供电话下单（以前全部邮寄 / 柜台）
- **1984**：并购 Bank of America's discount brokerage 业务
- **1987**：上市 NYSE
- **1989**：收购 Schwab 第一家 brick-and-mortar 连锁（"Schwab 500"分店）

### 1990-2000：互联网革命

- **1996**：推出 eSchwab，首家大规模**在线下单**券商
- **1999**：互联网券商黄金期，市值超过 Merrill Lynch
- **2000**：收购 U.S. Trust（私人银行）
- **2001 dot-com 崩盘**：Schwab 损失 60% 市值，裁员 1/3

### 2000-2020：大整合

- **2004**：Charles Schwab（创始人）复任 CEO 救公司
- **2010**：收购 optionsXpress（期权零售）
- **2017**：美股成交量突破 **零佣金**（Schwab 率先，Robinhood 模式的启发）
- **2019-10**：**Schwab 宣布永久零佣金**（股票 + ETFs + 期权）
- **2020**：**以 $26B 收购 TD Ameritrade**（加上 Pershing 整合）
- 完成后 Schwab **成为美国零售 broker 的**寡头之首

### 2020-2026：后整合期

- 2020-06 收购 TD Ameritrade 完成
- 2023 年整合 thinkorswim（原 TD 的期权平台，现 Schwab 主推）
- 2023 Q1 银行分支机构流动性危机（类似 SVB 小规模）—— Schwab Bank 面临 $41B 账户流出
- 2024 恢复 + 继续整合
- 2024 年成为美国 retirement account（401(k)）第二大提供者（仅次 Fidelity）

## 核心平台

### Schwab.com（Web）

- 主力零售 UI
- 适合大多数用户
- 包含账户管理 + 研究 + 交易 + 教育

### StreetSmart Edge + thinkorswim

- **thinkorswim**（原 TD Ameritrade 的平台）：被 Schwab 保留为高级版
- Windows + Mac + Mobile
- 期权 / 期货 / 复杂工具
- 行业标杆的零售期权交易平台

### Mobile apps

- Schwab Mobile（基础）
- Schwab All-In-One（整合 banking）
- thinkorswim Mobile（专业交易者）

## 业务模式：不靠 PFOF

### Schwab 的独特之处

和 Robinhood / TD Ameritrade 不同，Schwab 的营收**主要不是 PFOF**：

| 营收来源（2024）| 占比 |
|---|---|
| **银行 net interest income** | 55-60% |
| **资产管理费**（Schwab 基金 + 顾问服务）| 20-25% |
| **Trading 相关**（佣金 + 附加服务） | 10-15% |
| **PFOF + 其他** | **5-8%** |

**关键差异**：
- Schwab 有**自己的银行**（Charles Schwab Bank, FDIC 保险）
- 客户的**未投资现金**放 Schwab Bank 里，银行赚利差
- 这是比 PFOF 更稳定、更大规模的收入源
- **零佣金** 对 Schwab 不是"创新"，是"转移营收"（从佣金到利差）

### "零佣金"的实质

- 2019 Schwab 先降到零佣金 → 逼 TD Ameritrade、E*Trade 跟进 → 引发整合并购
- Schwab 不靠 PFOF 维持，靠银行 + 资管
- 但 Schwab **仍然做 PFOF**（只是占比小）
- Citadel + Virtu 是 Schwab 零售订单的主要做市商

## 与 Robinhood 对比

| 维度 | Charles Schwab | Robinhood |
|---|---|---|
| **创立** | 1971 | 2013 |
| **AUC** | $9.9T | $180B |
| **账户数** | 36M | 25M |
| **平均账户余额** | ~$275K | ~$4-6K |
| **用户画像** | 45+ 岁 + 富裕 | 22-35 岁 + 首次投资 |
| **主力营收** | 银行利差 55-60% | PFOF 60-70% |
| **产品** | 股 + 债 + 基金 + 期权 + 退休 + 银行 | 股 + 期权 + 加密 |
| **营销** | 低调 + 渠道 | 高调 + social |
| **OS 战略** | 综合服务 + 信任 | 零摩擦 + 游戏化 |
| **业内地位** | 寡头之首 | 新兴挑战者 |

**本质不同**：Schwab 是"美国富裕零售的托管方"，Robinhood 是"美国年轻人的第一次投资"。

## 独特优势

### 1. 银行化

- Schwab Bank 是**真正的银行**（FDIC 保险）
- 客户现金自动清扫到 Bank
- 客户可以同时银行 + 投资 + 贷款一家搞定
- 对竞争对手形成**结构性壁垒**

### 2. 养老金（401(k)）

- Schwab Retirement Plan Services 是美国第二大 401(k) 记录保管人
- 通过 401(k) 拉新 → 退休后变 rollover IRA
- 用户**终身粘性**

### 3. RIAs（注册投资顾问）

- Schwab 是美国最大的 **RIA custodian**（独立理财顾问的托管方）
- 大量独立顾问把客户资产放 Schwab
- 每个顾问带 $50M-$500M 资产
- 这是**机构级规模**的零售业务

## TD Ameritrade 整合（2020-2023）

### 整合细节

- 2020-06 完成并购（$26B 股票支付）
- 2020-2023 整合 TD 的 11M 客户到 Schwab 平台
- thinkorswim 平台保留（原 TD 的核心资产）
- 有些 TD 客户对 UI 变化不满 → 小部分流失
- 2023 Q4 整合基本完成

### 整合意义

- **美国零售 broker 寡头化**定音
- Schwab + Fidelity + Vanguard + IBKR 占美国零售 80%+
- 其他中小型 broker（E*Trade、Robinhood 等）**挑战寡头难度极大**

## 争议 / 挑战

### 2023 年 3 月"Schwab-crisis"

- 硅谷银行（SVB）事件时，Schwab Bank 也面临流动性担忧
- Schwab Bank 有 $41B 存款**外流**（客户转到高息货币市场基金）
- 股价当周跌 27%
- Schwab 用自有资本补救 + 合作伙伴流动性线
- **没有破产**，但暴露了银行化商业模式的**利率风险**

### 监管

- SEC + FINRA + CFTC（对期货部分）+ FDIC（银行部分）
- 多重监管下成本高但政治稳定
- 2023 年 Schwab 对 FINRA 进行 **$186M 和解**（与 TD 时代的 compliance 问题）
- 业务继续

### ESG + 政治

- Charles Schwab 创始人是政治保守派捐款者
- 公司在 "red state" politics 中被认为偏保守
- 年轻用户有偏好**更 progressive** 的平台（Fidelity、Vanguard）

## 未来方向

- **加密整合**（2023-2025 逐步）：Schwab 2025 Q2 推出 BTC + ETH ETFs 专属产品
- **AI 增强**：2024 推出 "Schwab Intelligent Portfolios" 的 AI 升级
- **国际**：2024-2026 扩张香港 + 新加坡（为 US-Asia 跨境零售）
- **继续整合**：预计未来 3-5 年还会收购中小 broker
